Über diese Demo

Willkommen zu dieser interaktiven E-Commerce-App für Fitness-Supplements! Aber bevor du dich ins Shopping-Abenteuer stürzt...

Bitte beachte, dass dies keine echte Website ist. Es handelt sich um eine Frontend-Demo, die zeigt, wie eine typische Online-Shop-App funktioniert. Es gibt keine echten Bestellungen, Zahlungen oder Daten - nur reine Funktionalität!

Vielen Dank, dass du diese Demo ausprobierst!

Gutscheincodes:

  • FITNESS20 - 20% Rabatt auf alle Produkte
  • SUPPLEMENT10 - 10% Rabatt auf alle Produkte
  • HEALTHY15 - 15% Rabatt auf Bestellungen über 100€
  • ENERGY25 - 25% Rabatt auf Energieriegel
  • GAINS30 - 30% Rabatt auf Weight Gainer

Technische Details:

  • Angular bildet die Grundlage der Anwendung, während Bootstrap für die Gestaltung und das responsive Design zuständig ist.
  • Für die Navigation wird Angular Routing verwendet, um zwischen verschiedenen Seiten und Komponenten zu wechseln - fast ohne Ladezeiten.
  • Der Warenkorb wird als Signal im CartService verwaltet. Die Quantität der einzelnen Produkte kann bearbeitet werden.
  • Gutscheincodes können beim Checkout eingelöst werden, dank Angular Signals sieht man den Rabatt sofort.
  • Der Warenkorb, das Checkout-Formular und die Gutscheincodes werden validiert. Der Benutzer erhält sofort Rückmeldungen, z. B. bei einem leeren Warenkorb, unvollständigem Formular oder Änderungen an Gutscheinen.
  • Die Darkmode-Funktion ist mit dem data-bs-theme-Attribut umgesetzt und kann mit einem Button gesteuert werden. Der Zustand wird im localStorage gespeichert, und beim ersten Besuch wird automatisch der Gerätezustand des Nutzers übernommen.
